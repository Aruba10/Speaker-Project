# AI Speaker Identification & Captioning

This project demonstrates a basic AI-driven pipeline for speaker-aware transcription.

It processes an audio file, splits it into segments, performs speech-to-text conversion, and tags each segment with speaker labels.

---

## Features
- Audio segmentation
- Automatic speech transcription
- Simulated speaker tagging
- Easy to run locally

---

## Project Structure
main.py → Main execution script  
sample.wav → Input audio file  
requirements.txt → Dependencies list

---

## Installation

1. Install Python
2. Install libraries

pip install -r requirements.txt

---

## Run Project

python main.py
